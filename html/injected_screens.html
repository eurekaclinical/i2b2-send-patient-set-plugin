Patient Set Sender

Introduction

Performs an action on an i2b2 patient set.

Instructions

  1. Navigate to the Select Patient Set tab.
  2. Drag and drop a patient set from Previous Queries or the Workplace onto the input box.
  3. Click the View Results tab to perform the selected action on the selected patient set.
